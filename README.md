# Task Manager Project

## Prerequisites
- Node.js (v18+)
- MongoDB (running locally on port 27017)

## Getting Started

### 1. Backend Setup

Navigate to the backend directory:
```bash
cd src/backend
```

Install dependencies:
```bash
npm install
```

Start the backend server:
```bash
npm run dev
```
The server will start on `http://localhost:5000`.

### 2. Frontend Setup

Navigate to the frontend directory (src root):
```bash
cd src
```

Install dependencies:
```bash
npm install
```

Start the frontend development server:
```bash
npm run dev
```
The application will be available at `http://localhost:5173`.

## Configuration

Both frontend and backend are configured to work with these default ports:
- Frontend: 5173
- Backend: 5000
- Database: MongoDB at `mongodb://localhost:27017/taskmanager`
